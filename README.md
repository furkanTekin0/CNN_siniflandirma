# CNN_siniflandirma
Ad: Furkan, Soyad: Tekin, Okul No: 2112721055, GitHub Repo: https://github.com/furkanTekin0/CNN_siniflandirma
. Bu projede kendi çektiğim görüntülerle (internet/veri seti indirimi yok) iki sınıflı bir CNN sınıflandırma çalışması yaptım: parfum (siyah parfüm şişesi) ve yuzuk (siyah yüzük). Her sınıf için 60 fotoğraf (toplam 120) topladım; farklı açı, ışık ve arka planlarda çekerek çeşitlilik sağladım ve eğitimde görüntüleri 128×128 (minimum 64×64) boyutuna yeniden ölçeklendirdim. Veri seti dataset/parfum/ ve dataset/yuzuk/ klasör yapısına uygun şekilde düzenlendi. model1.ipynb dosyasında ImageNet ağırlıklarıyla Transfer Learning (gerekirse fine-tuning) uygulayarak modeli eğittim ve eğitim/validasyon accuracy-loss grafikleri ile test doğruluğunu raporladım. model2.ipynb dosyasında CIFAR-10 benzeri basit CNN mimarisi kurup yine grafikleri ve test doğruluğunu verdim. model3.ipynb dosyasında Model2’yi geliştirerek en az 3 hiperparametreyi (filtre sayısı, batch size, öğrenme oranı, dropout, katman sayısı vb.) değiştirip ayrıca ImageDataGenerator ile data augmentation uyguladım; her denemenin test sonucunu tablo halinde sunup hangi değişikliklerin performansı artırdığını yorumladım. Notebook’lar sırasıyla model1 → model2 → model3 çalıştırılır; kurulum için pip install tensorflow matplotlib numpy pandas scikit-learn yeterlidir.
